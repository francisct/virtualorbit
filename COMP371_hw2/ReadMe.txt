
MY GRAVITATIONAL SYSTEM

======================================================================================================
DESCRIPTION
======================================================================================================

This software build scenes with lighting effects. The current scene is an gravitational
system similar to an atom.

======================================================================================================
CONTROLS
======================================================================================================

MOUSE:


Left click: Creates a cube. The longer you hold the click the bigger the cube.

Right click: Creates a sphere. The longer you hold the click the bigger the sphere.

Move pointer: Move the light




KEYBOARD:


F: Change lighting and freeze scene

UP, DOWN, LEFT, RIGHT ARROW: Move the world

E, Q: Zoom in/out

Z, X: Rotate world

W, A, S, D: Move core object (Player object)

O: Add platform between light and objects to create an esclipe

K, L: Change cam direction
